# 宜宾业务沟通纪要

## 一、新能源自卸车（渣土车）

### 基本信息
- **车辆性质**：公司自有车辆，非挂靠，员工制司机，有劳动合同
- **运营情况**：
  - 路线：临港白沙路 → 南溪方向（未到罗龙）
  - 单程约10-20公里，属临港新区范围
  - 已运营约2个月

### 利好因素
1. 短途运输
2. 固定路线/固定目的地
3. 公司化管理，非挂靠
4. 人保已承保（含单交），侧面印证赔付不高

## 二、集装箱运输车

### 基本信息
- **车型**：新能源小型集装箱车（短柜）
- **路线**：宜宾港 ↔ 犍为（乐山方向），走宜乐高速
- **频次**：每天最多2趟往返

### 利好因素
1. 固定线路
2. 高速为主，宜乐高速车流量小
3. 短途稳定业务

### 待确认
- 具体运输货物类型
- 是否涉及危化品（如是则不可承保）

## 三、货车承保底层逻辑（共识）

赔付低的货车特征：
1. 固定线路/固定目的地
2. 固定驾驶员
3. 公司化管理（非挂靠）
4. 业务稳定，无需长途奔波
5. 非过户车

## 五、系统/政策问题反馈

### 1. F类车定价问题
- 保费与平台一致，但竞争力不足
- 新能源影响大于燃油车

### 2. 过户车/首年投保
- **我司**：出险1次直接1.2倍系数
- **竞品**：0.8-0.9系数
- 首年投保（含停驶复驶）同样1.2倍，缺乏竞争力

### 3. 电销锁单问题
- 电销提前锁定客户，渠道无法报价
- 建议：至少开放报价权限

### 4. 流程时效
- 审批流程长，客户流失至其他公司
- 建议：优化特殊车型审批时效
