# 报表模板功能

## 概述

报表模板功能为用户提供预设的常用分析场景模板，帮助用户快速生成专业的数据分析报告。

## 功能特性

### 📊 模板分类
- **综合分析**: 月度经营汇总、业务总览等
- **业绩分析**: 业务员业绩、机构排名等
- **续保分析**: 续保率分析、到期提醒等
- **专项分析**: 营业货车、特定业务类型分析
- **增长分析**: 同比增长率、趋势分析等
- **对比分析**: 多维度数据对比分析

### 🎯 模板特点
- **预设配置**: 包含筛选条件、图表布局、KPI指标等
- **一键应用**: 点击模板即可快速配置看板
- **可自定义**: 在模板基础上进行个性化调整
- **移动端优化**: 响应式设计，支持移动设备

## 使用方法

1. 在看板中点击"报表模板"标签页
2. 选择合适的模板分类
3. 浏览模板描述和包含功能
4. 点击"使用此模板"应用配置
5. 根据需要调整自定义设置

## 技术实现

- **组件**: `ReportTemplatesPanel.tsx`
- **数据结构**: TypeScript接口定义模板结构
- **样式**: Tailwind CSS响应式设计
- **交互**: React状态管理处理模板选择
