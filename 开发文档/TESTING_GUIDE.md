## 🐛 常见问题排查

### 问题1: 上传后显示错误

**可能原因**: Schema验证失败

**检查**:
- 查看错误信息中提到的缺失字段
- 确认Parquet文件包含所有必需列

**解决**:
```python
# 重新生成Parquet，确保包含所有列
python3 << 'EOF'
# ... 使用前面的转换脚本 ...
EOF
```

### 问题2: KPI数值显示"..."

**可能原因**: 查询仍在执行

**检查**:
- 打开开发者工具Network标签
- 查看是否有Worker通信

**解决**: 等待几秒，如果长时间不变，刷新页面重试

### 问题3: 过滤器不生效

**可能原因**: 防抖延迟或JavaScript错误

**检查**:
- 控制台是否有错误
- 等待至少300ms

**解决**: 清空输入框，重新输入

### 问题4: 控制台报CORS错误

**可能原因**: DuckDB-WASM需要特定headers

**检查**: vite.config.ts中的headers配置

**解决**: 已在配置中添加，无需修改
