## 📋 新增字段概览

本次更新新增了6个业务分析字段，用于支持更深入的经营分析：

| 字段名 | 域字段 | 数据类型 | 业务用途 | 分析价值 |
|--------|----------|----------|----------|----------|
| 车架号 | vehicle_frame_no | STRING | 车辆唯一标识 | 支持车辆级精准追踪 |
| 是否报价 | is_quote | BOOL | 报价标志 | 计算报价转化率 |
| 赔案件数 | claim_cases | INT64 | 累计赔案数量 | 计算赔付率等KPI |
| 已报告赔款 | reported_claims | FLOAT64 | 累计赔款金额 | 盈利能力分析 |
| 费用金额 | fee_amount | FLOAT64 | 累计费用金额 | 成本结构分析 |
| 续保模式 | renewal_mode | STRING | 续保业务类型 | 续保策略效果分析 |

## 📊 业务分析能力扩展

### 赔付率分析
```sql
-- 赔付率 = 已报告赔款 / 保费收入
SELECT 
  SUM(reported_claims) / SUM(premium) as claim_ratio
FROM PolicyFact
WHERE premium > 0
```

### 费用率分析
```sql
-- 费用率 = 费用金额 / 保费收入
SELECT 
  SUM(fee_amount) / SUM(premium) as fee_ratio
FROM PolicyFact
WHERE premium > 0
```

### 综合成本率分析
```sql
-- 综合成本率 = (已报告赔款 + 费用金额) / 保费收入
SELECT 
  SUM(reported_claims + fee_amount) / SUM(premium) as comprehensive_cost_ratio
FROM PolicyFact
WHERE premium > 0
```

### 边际贡献率分析
```sql
-- 边际贡献率 = (保费收入 - 已报告赔款 - 费用金额) / 保费收入
SELECT 
  SUM(premium - reported_claims - fee_amount) / SUM(premium) as contribution_ratio
FROM PolicyFact
WHERE premium > 0
```

### 报价转化率分析
```sql
-- 报价转化率 = COUNT(是否报价=True) / COUNT(*)
SELECT 
  COUNT(CASE WHEN is_quote = True THEN 1 END) / COUNT(*) as quote_conversion_rate
FROM PolicyFact
```

### 续保模式效果分析
```sql
-- 续保模式分布与电销转化
SELECT 
  renewal_mode,
  COUNT(*) as policy_count,
  COUNT(CASE WHEN is_telemarketing = True THEN 1 END) as telemarketing_count,
  COUNT(CASE WHEN is_telemarketing = True THEN 1 END) / COUNT(*) as telemarketing_rate
FROM PolicyFact
WHERE renewal_mode IS NOT NULL
GROUP BY renewal_mode
```
