## 裁决后流程

1. 你填完 → 我按裁决逐条改对应 SQL 生成器/注册表。
2. 每条改完用 `duckdb` 直查 Parquet 与 API 返回对比（红线 §6 源数据验证）。
3. 与公司报表对账（赔付率类容忍 <0.2 个百分点）。
4. 注册表口径变更同步 `version` + `changelog`，跑 `bun scripts/metric-registry/validate.ts`。
5. 一批一 PR，BACKLOG 对应项标 DONE。
