## 成本指标

### 满期赔付率 (`earned_claim_ratio`)

- **版本**: 2.0.0
- **标签**: core, kpi, cost
- **单位**: %
- **公式**: 已报告赔款 / 满期保费（闰年感知）
  - 分子: SUM(reported_claims)
  - 分母: SUM(premium * earned_days / policy_term)
- **展示**: 赔付率 (percent, %)
- **注意**: 赔案口径：report_time < 观察截止（MAX(report_time)），已结案(settlement_time<观察点)取settled_amount，否则取reserve_amount。保单口径：按policy_no聚合净保费>0。分母闰年感知：policy_term = DATEDIFF(起期, 起期+1年) = 365或366；earned_days = MIN(已过天数, policy_term)，退保保单截止于退保日

```sql
CASE
    WHEN SUM(premium * CAST(earned_days AS DOUBLE) / CAST(policy_term AS DOUBLE)) > 0
    THEN ROUND(SUM(reported_claims) * 100.0 / SUM(premium * CAST(earned_days AS DOUBLE) / CAST(policy_term AS DOUBLE)), 2)
    ELSE NULL
  END AS earned_claim_ratio
```

**变更历史**:
- v1.0.0 (2026-03-27): 从 cost.ts 迁移
- v1.1.0 (2026-04-11): 口径修正：赔案锚点改为 report_time，已决/未决按 settlement_time 分类取值，保单净保费聚合排除完全退保，截止日期改为 MAX(report_time)
- v2.0.0 (2026-04-17): 铁律对齐：分母从 exposure_days/365 改为 earned_days/policy_term（闰年感知）；展示精度 2→1 位小数

### 基准保费 (`baseline_premium`)

- **版本**: 1.0.0
- **标签**: cost, pricing, baseline
- **单位**: 元
- **公式**: 基准保费：商业险按商车自主定价系数归一，非商业险保持原保费
  - 分子: 商业险 premium / commercial_pricing_factor；非商业险 premium
- **展示**: 基准保费 (premiumWan, 万元)
- **注意**: 先计算基准保费，再进入满期口径。商业险自主系数缺失、为0或负数时输出 NULL，调用方应单独输出异常数据；交强险等非商业险不做自主系数折算。

```sql
CASE
    WHEN insurance_type = '商业保险' AND commercial_pricing_factor > 0
    THEN premium / NULLIF(commercial_pricing_factor, 0)
    WHEN insurance_type = '商业保险'
    THEN NULL
    ELSE premium
  END AS baseline_premium
```

**变更历史**:
- v1.0.0 (2026-05-12): 新增：商业险价格因子归一的基准保费，作为满期基准保费的前置层

### 满期基准保费 (`baseline_earned_premium`)

- **版本**: 1.0.0
- **标签**: cost, pricing, baseline
- **单位**: 元
- **公式**: 满期基准保费 = 基准保费 × 满期天数 / 保险期限天数（闰年感知）
  - 分子: SUM(baseline_premium * earned_days / policy_term)
- **展示**: 满期基准保费 (premiumWan, 万元)
- **注意**: 自包含表达式：内联 baseline_premium CASE 以避免别名引用（domain-testcases 测试单独消费）。对于已完全满期保单，earned_days = policy_term，满期基准保费等于基准保费。

```sql
ROUND(SUM(
    (CASE
      WHEN insurance_type = '商业保险' AND commercial_pricing_factor > 0
      THEN premium / NULLIF(commercial_pricing_factor, 0)
      WHEN insurance_type = '商业保险'
      THEN NULL
      ELSE premium
    END) * CAST(earned_days AS DOUBLE) / CAST(policy_term AS DOUBLE)
  ), 2) AS baseline_earned_premium
```

**变更历史**:

### 满期基准赔付率 (`baseline_earned_claim_ratio`)

- **版本**: 1.0.0
- **标签**: cost, pricing, baseline
- **单位**: %
- **公式**: 满期基准赔付率 = 已报告赔款 / 满期基准保费
  - 分子: SUM(reported_claims)
  - 分母: SUM(baseline_earned_premium)
- **展示**: 满期基准赔付率 (percent, %)
- **注意**: 自包含表达式：内联满期基准保费分母以避免别名引用。分子分母使用一致的有效样本过滤（商业险无效自主系数记录在分子分母都排除）。

```sql
CASE
    WHEN SUM(
      (CASE
        WHEN insurance_type = '商业保险' AND commercial_pricing_factor > 0
        THEN premium / NULLIF(commercial_pricing_factor, 0)
        WHEN insurance_type = '商业保险'
        THEN NULL
        ELSE premium
      END) * CAST(earned_days AS DOUBLE) / CAST(policy_term AS DOUBLE)
    ) > 0
    THEN ROUND(SUM(
      CASE
        WHEN insurance_type = '商业保险' AND NOT (commercial_pricing_factor > 0)
        THEN NULL
        ELSE reported_claims
      END
    ) * 100.0 / SUM(
      (CASE
        WHEN insurance_type = '商业保险' AND commercial_pricing_factor > 0
        THEN premium / NULLIF(commercial_pricing_factor, 0)
        WHEN insurance_type = '商业保险'
        THEN NULL
        ELSE premium
      END) * CAST(earned_days AS DOUBLE) / CAST(policy_term AS DOUBLE)
    ), 2)
    ELSE NULL
  END AS baseline_earned_claim_ratio
```

### 变动成本率 (`variable_cost_ratio`)

- **版本**: 2.0.0
- **标签**: core, kpi, cost
- **单位**: %
- **公式**: 满期赔付率 + 费用率（两个分母不同；闰年感知）
  - 分子: 已报告赔款/满期保费 + 费用金额/签单保费
- **展示**: 变动成本率 (percent, %)
- **注意**: 赔付率分母=满期保费（premium × earned_days / policy_term，闰年感知），费用率分母=签单保费。可超100%（亏损）

```sql
CASE
    WHEN SUM(premium * CAST(earned_days AS DOUBLE) / CAST(policy_term AS DOUBLE)) > 0 AND SUM(premium) > 0
    THEN ROUND(
      SUM(reported_claims) * 100.0 / SUM(premium * CAST(earned_days AS DOUBLE) / CAST(policy_term AS DOUBLE)) +
      SUM(COALESCE(fee_amount, 0)) * 100.0 / SUM(premium),
      2
    )
    ELSE NULL
  END AS variable_cost_ratio
```

**变更历史**:
- v1.0.0 (2026-03-27): 新增，与 cost.ts:generateVariableCostQuery 一致

### 满期出险率 (`earned_loss_frequency`)

- **版本**: 2.1.0
- **标签**: cost
- **单位**: %
- **公式**: (赔案件数/保单件数) × (保险期限天数/满期天数)。满期后=赔案/保单，未满期年化放大
  - 分子: SUM(claim_cases × policy_term / earned_days)
  - 分母: COUNT(DISTINCT policy_no)
- **展示**: 出险率 (percent, %)
- **注意**: 闰年感知：policy_term = DATEDIFF(起期, 起期+1年) = 365或366天。earned_days = MIN(已过天数, policy_term)。满期后 ratio=1，未满期 ratio>1

```sql
CASE
    WHEN COUNT(DISTINCT policy_no) > 0 AND SUM(earned_days) > 0
    THEN ROUND(
      SUM(claim_cases * 1.0 * policy_term / NULLIF(earned_days, 0))
      / COUNT(DISTINCT policy_no) * 100.0,
      2
    )
    ELSE NULL
  END AS earned_loss_frequency
```

**变更历史**:
- v1.0.0 (2026-03-27): 从 cost.ts 迁移
- v2.0.0 (2026-03-31): 口径修正：保单级→年化公式，闰年感知(365/366)

  - 分子: earned_premium × (1 - earned_claim_ratio - expense_ratio)
- **展示**: 满期边际贡献额 (premiumWan, 万元)
- **注意**: 基于已赚保费的实际边际贡献，随满期天数增长而变化。已满期保单与 projected_margin_amount 相等。闰年感知分母

```sql
CASE
    WHEN SUM(premium * CAST(earned_days AS DOUBLE) / CAST(policy_term AS DOUBLE)) > 0 AND SUM(premium) > 0
    THEN ROUND(
      SUM(premium * CAST(earned_days AS DOUBLE) / CAST(policy_term AS DOUBLE)) * (
        1.0
        - SUM(reported_claims) / SUM(premium * CAST(earned_days AS DOUBLE) / CAST(policy_term AS DOUBLE))
        - SUM(COALESCE(fee_amount, 0)) / SUM(premium)
      ), 2
    )
    ELSE NULL
  END AS earned_margin_amount
```
