## 成本指标

### 满期赔付率 (`earned_claim_ratio`)

- **版本**: 1.0.0
- **标签**: core, kpi, cost
- **单位**: %
- **公式**: 已报告赔款 / 满期保费
  - 分子: SUM(reported_claims)
  - 分母: SUM(premium * exposure_days / 365)
- **展示**: 赔付率 (percent, %)
- **注意**: 需在 CTE 中预算 exposure_days = LEAST(GREATEST(DATEDIFF(day, 起保日, 截止日), 0), 365)

```sql
CASE
    WHEN SUM(premium * CAST(exposure_days AS DOUBLE) / 365.0) > 0
    THEN ROUND(SUM(reported_claims) * 100.0 / SUM(premium * CAST(exposure_days AS DOUBLE) / 365.0), 2)
    ELSE NULL
  END AS earned_claim_ratio
```

### 变动成本率 (`variable_cost_ratio`)

- **版本**: 1.0.0
- **标签**: core, kpi, cost
- **单位**: %
- **公式**: 满期赔付率 + 费用率（注意：两个分母不同）
  - 分子: 已报告赔款/满期保费 + 费用金额/签单保费
- **展示**: 变动成本率 (percent, %)
- **注意**: 赔付率分母=满期保费，费用率分母=签单保费。可超100%（亏损）

```sql
CASE
    WHEN SUM(premium * CAST(exposure_days AS DOUBLE) / 365.0) > 0 AND SUM(premium) > 0
    THEN ROUND(
      SUM(reported_claims) * 100.0 / SUM(premium * CAST(exposure_days AS DOUBLE) / 365.0) +
      SUM(COALESCE(fee_amount, 0)) * 100.0 / SUM(premium),
      2
    )
    ELSE NULL
  END AS variable_cost_ratio
```
