# 摩托车经营诊断（v1.0）

> 摩托车 = 交强险120元/件 + 捆绑人身险。**不能只看交强赔付率**，必须用完整成本模型算真实盈亏。
> 管理层三问：**算上人身险赚不赚？哪些细分在赚？能不能扩大盈利池？**

---

## 摩托车业务本质（RED LINE）

```
摩托车交强险 ≠ 普通车险

1. 交强险120元/件，费率监管制定，无法调价
2. 每单捆绑销售人身险（意外险），这才是利润来源
3. 车险费用率 = 0（交强险不支付渠道佣金）
4. 人身险通过佣金激励中介销售交强险
```

**禁止事项**：
- ❌ 建议"调价/提高定价系数/风险评分定价/核保收紧"
- ❌ 用100%作为盈亏平衡线（综合成本率框架下 100% 才是盈亏线）
- ❌ 忽略人身险利润只看交强赔付率

**应聚焦**：
- ✅ 用完整成本模型(交强+人身-佣金-管理费)算真实盈亏
- ✅ 按维度找盈利/亏损细分
- ✅ 渠道管理（经代质量筛选）
- ✅ 承保选择（品牌/车型/车龄白黑名单）

## 完整成本模型

### 机构分类

| 类型 | 机构 | 人身险保费 | 人身险佣金率 |
|------|------|-----------|------------|
| **A类** | 宜宾、泸州、自贡 | 200元/件 | 65% |
| **B类** | 天府、乐山、高新、武侯、德阳、资阳、达州、新都等 | 100元/件 | 60% |

### 成本公式

```
每单成本 = 交强赔付 + 人身险佣金 + 人身险赔付(5%) + 销售推动费 + 管理费
  销售推动费 = 交强险×0.15% + 人身险×0.74%（A类1.66元，B类0.92元）
  管理费 = 总收入 × 管理成本率（各机构不同）
盈亏平衡赔付率 = (总收入 - 人身险佣金 - 人身险赔付 - 推动费 - 管理费) / 120
```

> 不计算员工工资（管理成本率已覆盖）。销售推动费不是人力费用，单独保留。

## Phase 2: 成本模型研判

**读取机构下钻 JSON 后，必须执行成本模型计算**：

### Step 1: 分类机构
```python
A_TYPE = {'宜宾', '泸州', '自贡'}  # 200元人身险, 佣金65%
B_TYPE = {其余所有机构}              # 100元人身险, 佣金60%
```

### Step 2: 逐机构计算真实盈亏
```
对每个机构:
  1. 判断 A/B 类 → 确定人身险保费和佣金率
  2. 计算变动成本（人身险佣金+赔付+工资+推动费）
  3. 计算管理费（总保费×7.5%，或用机构专属费率）
  4. 净利润 = 总收入 - 交强赔付 - 变动成本 - 管理费
  5. 单均净利 = 净利润 / 保单数
  6. 判定: 赔付率 vs 盈亏线(A:123.7% / B:111.5%)
```

### Step 3: 输出成本模型表
必须包含：机构、类型、保单数、交强赔付率、盈亏线、裕度、净利润、单均净利、判定。

## Phase 3: 多维毒性分析（按需触发）

### 标准9维度
被保险人年龄、车牌归属地、车型(按排量)、新旧车、车价、车龄、业务员、经代、能源

### 毒性指数
```
毒性指数 = 赔付贡献度% / 保费占比%
```
- **> 1.3**：毒瘤（保费小赔付大）
- **< 0.8**：盈利池

### 盈亏判定必须用真实盈亏线

对每个维度的每个值，判定盈亏时：
- 不能用100%
- 用加权平均盈亏线 = A类保费占比×123.7% + B类保费占比×111.5%
- 或按该维度值在各机构的分布，分别计算

## 指标体系

与 `/diagnose-agent` 相同，但增加：

| 指标 | 公式 |
|------|------|
| 综合收入/单 | 交强险120 + 人身险(200或100) |
| 综合变动成本/单 | 交强赔付 + 人身险佣金 + 人身险赔付 + 工资 + 推动费 |
| 管理费/单 | 综合收入 × 管理费用率(默认7.5%) |
| **单均净利** | 综合收入 - 综合变动成本 - 管理费 |
| **真实盈亏平衡赔付率** | (综合收入 - 非赔付变动成本 - 管理费) / 120 |

## 亮灯规则（用各机构实际盈亏线）

|------|--------|--------|--------|--------|
| A类赔付率 | ≤115% | 115-125% | 125-盈亏线 | >盈亏线 |
| B类赔付率 | ≤100% | 100-110% | 110-盈亏线 | >盈亏线 |
