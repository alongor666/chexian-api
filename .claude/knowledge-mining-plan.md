## 🎯 挖掘执行计划

### Step 1: 对话扫描 (5分钟)
```
执行者: AI
操作:
1. 扫描本次对话完整记录
2. 识别所有关键词命中
3. 提取上下文片段
4. 生成候选知识清单

输出: 候选知识清单 (X条)
```

### Step 2: 初步分类 (5分钟)
```
执行者: AI
操作:
1. 对候选知识进行分类 (A-F类)
2. 提取规则/纠正/约束的原始内容
3. 生成规范化表述
4. 标记不确定的知识点

输出: 分类知识清单 (A类X条, B类Y条, ...)
```

### Step 3: 逐项确认 (15-30分钟)
```
执行者: 您 + AI
操作:
1. AI逐项输出理解
2. 您确认/修正/补充/丢弃
3. AI更新规则
4. 重复直到所有项目确认

输出: 确认知识清单 (Y条)
```

### Step 4: 知识归档 (10分钟)
```
执行者: AI
操作:
1. 按知识类型选择存储位置
2. 更新对应文档文件
3. 添加交叉引用
4. 生成变更摘要

输出: 知识库更新 (新增Y条, 修改Z条)
```

### Step 5: 质量检查 (5分钟)
```
执行者: AI
操作:
1. 检查规则冲突
2. 验证代码一致性
3. 检查文档完整性
4. 生成健康度报告

输出: 知识库健康度报告
```
