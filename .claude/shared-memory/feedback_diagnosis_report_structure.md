经营诊断报告标准 7 板块结构：

1. **整体经营概况** — 按年份展开，最后一列"趋势分析"为 AI 文字点评
2. **新转续过户维度** — 2.0 汇总表 + 2.1-2.4 各类分年展开
3. **能源类型** — 非新-燃/非新-天/新能源，无数据项预留空列
4. **风险评分** — 智能检测评分字段 + "无评分"列
5. **季度趋势** — 汇总表（21Q1 格式）+ 7 个分项 ASCII 条形图
6. **险类** — 商业险/交强险汇总 + 分年展开
7. **诊断总结** — 年度亮灯总结 + 关键发现 + 更多维度建议

**每个表的指标行顺序**（置顶到底部）：
满期边际贡献额 → 预估边际贡献额 → 变动成本率 → 边际贡献率 → 赔付组 → 出险组 → 费用组 → 保费组 → 系数组

**趋势分析列**：AI 文字，如"大幅上升↑ 高2022:135.0 低2026:50.8"
**维度分析列**：如"高:旧车过户 低:旧车续保"

**Why:** 用户需要标准化的报告格式，便于不同车型/机构/经代的横向对比。

**How to apply:** 新建诊断类报告时，先用 diagnose_vehicle.py 生成标准 7 板块，再按需增减维度。
