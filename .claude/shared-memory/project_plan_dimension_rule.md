name: 保费计划维度限制
description: 只有分公司/三级机构/销售团队/业务员有保费计划数据，其他维度不显示计划达成率指标
type: project
---

保费计划数据仅存在于分公司/三级机构/销售团队/业务员维度。

**Why:** 保费计划是按组织架构层级分配的（公司→机构→团队→业务员），客户类别、险种组合等业务维度没有对应的计划数据。

**How to apply:**
- 涉及计划达成率的图表/指标，当筛选维度不在 {company, org, team, salesman} 时，不显示达成率相关系列
- 在趋势图等组合图中，达成率折线应根据是否有计划数据动态显示/隐藏
- `achievement_cache` 表仅包含有计划的业务员数据，查询时注意 NULL/空值处理
