Parquet 中有三个风险评分字段，按客户类别互斥覆盖：
- `车险风险等级`：非营业客车（覆盖最广）
- `大货车评分`：营业货车-大货车/牵引车
- `小货车评分`：营业货车-小货车

三字段值域统一 A-G/X，互不重叠（同一条记录最多一个有值）。

**智能检测逻辑**（diagnose_vehicle.py `detect_risk_field()`）：
1. COUNT 三个字段的非空覆盖率
2. 按覆盖率降序排列
3. 返回 `COALESCE(覆盖最高, 次高, ...)` 表达式

**牵引车实测**：车险风险等级 7,617 件 + 大货车评分 3,676 件 = 11,293 件（63%），剩余 6,630 件无评分（需单独列"无评分"列）。

**Why:** 不同客户类别使用不同评分字段，硬编码任一字段会丢失数据。

**How to apply:** 所有涉及风险评分的查询/报告，用 `detect_risk_field()` 自动检测，不要硬编码字段名。
