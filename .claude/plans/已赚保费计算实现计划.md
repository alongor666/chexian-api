## 实现方案

### 1. 已赚保费计算公式

根据用户提供和Excel验证的公式：

```
已赚保费 = 险类保费 × 费用率 × 险类系数
         + 险类保费 × (1 - 费用率) × (满期天数/365)
```

**参数说明**:
- **费用率**: 费用金额 / 保费（每张保单实际值）
- **险类系数**: 交强险 = 0.8223, 商业保险 = 0.9408
- **满期天数**: 从保险起期起算，当日为1，次日为2，...，第365天为365

### 2. 两段式已赚保费计算（会计年度制）

**用户确认的处理规则**:
- ✅ 2025年保单：计算2025年末和2026年3月末的已赚保费
- ✅ 2026年保单：只计算2026年3月末的已赚保费
- ✅ 2025年以前保单：不计算（过滤掉）
- ✅ 异常值处理：保留但标记（添加"数据质量"字段）

#### 第一段：2025年末已赚保费（仅2025年保单）
- **计算基准日**: 2025-12-31
- **满期天数计算**: 从保险起期到2025-12-31的天数
- **适用场景**: 2025年度利润考核

#### 第二段：2026年3月末已赚保费
- **2025年保单**: 从保险起期到2026-03-31的天数
- **2026年保单**: 从保险起期到2026-03-31的天数
- **适用场景**: 2026年度Q1利润考核

### 4. 实现步骤

#### Step 1: 数据预处理
```python
# 读取parquet文件
df = pd.read_parquet('车险保单综合明细表.parquet')

# 提取保单年度和月份
df['保单年度'] = df['签单日期'].dt.year
df['保单月份'] = df['签单日期'].dt.month

# 计算费用率
df['费用率'] = df['费用金额'] / df['保费']

# 数据质量标记（保留异常但标记）
df['数据质量'] = '正常'
df.loc[df['保费'] <= 0, '数据质量'] = '保费负值'
df.loc[df['费用率'] > 1, '数据质量'] = '费用率异常'
df.loc[df['费用率'] < 0, '数据质量'] = '费用率异常'

# 设置险类系数
df['险类系数'] = df['险类'].map({
    '交强险': 0.8223,
    '商业保险': 0.9408
})

# 只保留2025和2026年保单
df = df[df['保单年度'].isin([2025, 2026])]
```

#### Step 2: 计算满期天数
```python
# 定义基准日
date_2025_end = pd.Timestamp('2025-12-31')
date_2026_q1_end = pd.Timestamp('2026-03-31')

# 计算满期天数函数
def calculate_maturity_days(start_date, end_date):
    days = (end_date - start_date).days + 1  # 当日为1
    return min(max(days, 0), 365)  # 限制在0-365之间

# 2025年末满期天数（仅2025年保单）
df['满期天数_2025末'] = df.apply(
    lambda row: calculate_maturity_days(row['保险起期'], date_2025_end)
    if row['保单年度'] == 2025 else 0,
    axis=1
)

# 2026年Q1末满期天数（所有保单）
df['满期天数_2026Q1末'] = df['保险起期'].apply(
    lambda x: calculate_maturity_days(x, date_2026_q1_end)
)
```

#### Step 3: 计算已赚保费
```python
# 已赚保费计算函数
def calculate_earned_premium(row, days_col):
    premium = row['保费']
    fee_rate = row['费用率']
    coeff = row['险类系数']
    days = row[days_col]

    # 第一部分：首日费用（一次性计入）
    part1 = premium * fee_rate * coeff

    # 第二部分：时间分摊
    part2 = premium * (1 - fee_rate) * (days / 365)

    return part1 + part2

# 2025年末已赚保费（仅2025年保单）
df['已赚保费_2025末'] = df.apply(
    lambda row: calculate_earned_premium(row, '满期天数_2025末')
    if row['保单年度'] == 2025 else 0,
    axis=1
)

# 2026年Q1末已赚保费（所有保单）
df['已赚保费_2026Q1末'] = df.apply(
    lambda row: calculate_earned_premium(row, '满期天数_2026Q1末'),
    axis=1
)
```

#### Step 4: 输出CSV
```python
# 选择输出字段
output_columns = [
    '三级机构', '保单年度', '保单月份', '险类', '保费',

    '满期天数_2025末', '满期天数_2026Q1末',
    '已赚保费_2025末', '已赚保费_2026Q1末', '数据质量'
]

# 输出到当前目录
output_path = './已赚保费计算表.csv'
df[output_columns].to_csv(output_path, index=False, encoding='utf-8-sig')

# 输出统计信息
print(f"总保单数: {len(df)}")
print(f"2025年保单: {len(df[df['保单年度'] == 2025])}")
print(f"2026年保单: {len(df[df['保单年度'] == 2026])}")

print(df['数据质量'].value_counts())
print(f"\n已赚保费统计:")
print(f"  2025年末总计: {df['已赚保费_2025末'].sum():.2f}")
print(f"  2026年Q1末总计: {df['已赚保费_2026Q1末'].sum():.2f}")
```

## 验证计划

### 数据验证
- 检查费用率是否在合理范围（0-1，异常值标记但不过滤）
- 检查满期天数是否在0-365之间
- 检查已赚保费是否 ≤ 保费（正常情况下）
- 抽样验证：手工计算10条记录，对比程序结果
- 检查数据质量标记分布

### 业务验证
- 2025年保单：已赚保费_2025末 ≤ 已赚保费_2026Q1末

- 交强险系数0.8223应用正确

## 风险与注意事项

1. **费用率异常值**: 存在费用率>1或<0的情况，已通过"数据质量"字段标记，保留但不影响计算
2. **保费负值**: 存在保费为负的记录，已通过"数据质量"字段标记
3. **数据过滤**: 2025年以前保单已过滤，只保留2025和2026年保单
