# 商车自主定价系数监控板块开发计划

## 1. 需求概述

新增"系数监控"板块，位于增长率分析Tab之后，用于监控商车自主定价系数。

### 1.1 核心计算逻辑
```
NCD保费 = 保费 / 商车自主定价系数（仅限商业险）
聚合商车自主定价系数均值 = 聚合保费 / 聚合NCD保费
```

### 1.2 表格字段（最终版）
| 字段 | 说明 | 数据来源 |
|------|------|----------|
| 地域 | 成都/全省/具体三级机构 | 聚合逻辑 |
| 是否新能源 | 新能源/燃油 | is_nev |
| 客户类别 | 非营业个人客车/全部 | customer_category |
| 是否新车 | 新车/旧车 | is_new_car |
| 险类 | 商业险（固定） | insurance_type = '商业保险' |
| 当天系数 | 当日均值 | 计算 |
| 当周系数 | 当周均值（特殊周期适用时不同） | 计算 |
| 当月系数 | 当月均值 | 计算 |
| 当年系数 | 当年均值 | 计算 |
| 监管阈值 | 成都/全省规则，异地待定 | 配置 |
| 阈值方向 | ≥ 或 ≤ | 配置 |
| **当周与阈值差** | **当周系数 - 监管阈值（减法，非除法）** | 计算 |
| **缺口保费** | **(监管阈值 - 当周系数) × 当周保费** | 计算 |
| 是否合规 | 根据阈值方向判断 | 计算 |

### 3.2 核心SQL逻辑
```sql
-- 计算聚合系数均值
SELECT
  org_level_3,
  is_nev,
  customer_category,
  is_new_car,
  '商业保险' AS insurance_type,
  SUM(premium) AS total_premium,
  SUM(premium / commercial_pricing_factor) AS total_ncd_premium,
  SUM(premium) / NULLIF(SUM(premium / commercial_pricing_factor), 0) AS avg_factor
FROM PolicyFact
WHERE insurance_type = '商业保险'
  AND commercial_pricing_factor IS NOT NULL
  AND commercial_pricing_factor > 0
GROUP BY org_level_3, is_nev, customer_category, is_new_car
```

### 3.3 周期计算逻辑
```typescript
function getWeekPeriod(date: Date, isSpecialCase: boolean): { start: Date, end: Date } {
  const day = date.getDate();
  const year = date.getFullYear();
  const month = date.getMonth();
  const lastDay = new Date(year, month + 1, 0).getDate();

  if (isSpecialCase) {
    // 新能源非营业个人客车旧车: 1-14, 15-月末
    if (day <= 14) {
      return { start: new Date(year, month, 1), end: new Date(year, month, 14) };
    } else {
      return { start: new Date(year, month, 15), end: new Date(year, month, lastDay) };
    }
  } else {
    // 一般情况: 1-7, 8-14, 15-21, 22-月末
    if (day <= 7) {
      return { start: new Date(year, month, 1), end: new Date(year, month, 7) };
    } else if (day <= 14) {
      return { start: new Date(year, month, 8), end: new Date(year, month, 14) };
    } else if (day <= 21) {
      return { start: new Date(year, month, 15), end: new Date(year, month, 21) };
    } else {
      return { start: new Date(year, month, 22), end: new Date(year, month, lastDay) };
    }
  }
}
```

## 7. 性能优化记录

### 7.1 N+1 查询问题已解决 ✅
