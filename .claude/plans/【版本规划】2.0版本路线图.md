## 🎯 需求0：分析图表视角切换（5个任务）

### 背景
现有系统仅支持保费视角分析，无法满足按保单件数分析的需求（如商业险件数、交强险件数）。

### 实现方案

#### B109: 架构设计（P0，基础架构）
**交付物**：
- `src/shared/types/view-perspective.ts` - 视角类型定义
- `src/features/dashboard/hooks/usePerspective.ts` - 全局状态管理
- `src/shared/sql/perspective-adapter.ts` - SQL生成器适配层
- `src/widgets/filters/PerspectiveSwitcher.tsx` - 标签切换UI组件

**核心类型**：
```typescript
type ViewPerspective = 'premium' | 'commercial_count' | 'compulsory_count';

interface PerspectiveConfig {
  type: ViewPerspective;
  label: string; // 保费 | 商业险件数 | 交强险件数
  aggregation: 'SUM(premium)' | 'COUNT(*) WHERE 险类=商业险' | 'COUNT(*) WHERE 险类=交强险';
  yAxisLabel: string; // 保费（万元）| 件数 | 件数
}
```

**架构原则**：
- KPI看板不受视角影响（始终显示保费）
- 所有其他分析图表支持视角切换
- 视角状态全局共享（避免每个图表独立状态）

#### B110: 趋势分析视角切换（P0，核心功能）
**改造范围**：
- `generateDailyTrendQuery` - 日趋势SQL支持视角参数
- `generateWeeklyTrendQuery` - 周趋势SQL支持视角参数
- `generateMonthlyTrendQuery` - 月趋势SQL支持视角参数
- `TrendSection` - 集成 PerspectiveSwitcher
- `LineChart` - 动态Y轴标签（保费万元/件数）

**SQL示例**（保费视角）：
```sql
SELECT
  date_key,
  SUM(premium) AS total_value
FROM PolicyFact
GROUP BY date_key
```

**SQL示例**（商业险件数视角）：
```sql
SELECT
  date_key,
  COUNT(*) AS total_value
FROM PolicyFact
WHERE 险类 = '商业险'
GROUP BY date_key
```

#### B111-B113: 其他分析板块视角切换（P1-P2）
- **B111**: 营业货车分析（吨位分段+机构维度）
- **B112**: 增长率分析（同比/环比/累计）
- **B113**: 续保分析（续保率需适配视角）

### 实现方案

#### 两行置顶布局设计

**第一行（核心筛选，始终可见）**：

```

**第二行（维度筛选，始终可见）**：
```
[三级机构▼] [客户类别▼] [险别组合▼]
```

**折叠区域（高级筛选）**：
```

▼ 快捷组合（默认展开）
  [同城业务] [异地业务] [可续业务] [套单业务] ...

## 📊 需求2：优质业务分析增强（4个任务）

### 背景
现有优质业务分析可折叠，用户容易忽略；业务员明细Top100信息密度过高。

### 实现方案

#### B118: 固定显示（P1）
- 移除 `showQualityBusinessTrend` 状态
- 移除折叠按钮
- 优质业务板块始终显示
- 固定位置：趋势分析下方，营业货车上方

#### B119: X轴优化（P1）
**需求**：日期数量<32天时，X轴只显示首尾日期（MM-DD格式）

**实现逻辑**：
```typescript
axisLabel: {
  formatter: (value, index) => {
    if (data.length >= 32) return value; // 默认显示
    if (index === 0) return formatMMDD(data[0].date); // 首日
    if (index === data.length - 1) return formatMMDD(data[data.length - 1].date); // 末日
    return ''; // 其他隐藏
  }
}
```

#### B120: 值标签固定显示（P1）
**配置**：
```typescript
series: {
  label: {
    show: true, // 强制显示
    formatter: '{c}%', // 显示占比
    position: 'top'
  }
}
```

#### B121: 业务员双表排名（P1）
**布局**：
```
┌─────────────────────────────┬─────────────────────────────┐
│ 全部业务 Top10              │ 优质业务 Top10              │
│ 排名 业务员 总保费          │ 排名 业务员 优质业务保费    │
│ 1    张三   1000万          │ 1    李四   800万           │
│ 2    李四   900万           │ 2    张三   700万           │
│ ...                         │ ...                         │
└─────────────────────────────┴─────────────────────────────┘
```

**SQL查询**：
- `generateSalesmanAllBusinessRankingQuery` - 全部业务Top10
- `generateSalesmanQualityBusinessRankingQuery` - 优质业务Top10
