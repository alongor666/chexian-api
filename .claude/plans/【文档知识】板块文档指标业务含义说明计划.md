### 03_变动成本板块.md

#### 需要修改的位置

**双轴散点图系统（第13-17行）**
- 当前状态：4个指标只有公式
- 修改方案：

```markdown
- **核心指标**:

  1. **满期赔付率** (X轴)
     - 计算公式: 已报告赔款 / 满期保费 × 100%
     - 业务含义: 衡量保费中被赔付消耗的比例，反映承保风险高低
     - 解读指南:
       * ≤70%: 承保质量优秀
       * 70%-75%: 需关注赔付控制
       * >75%: 赔付率过高，需改善风险选择
     - 单位: 百分比（%），范围0-130
     - 应用场景: 评估承保端的风险管理能力，识别高风险维度
     - 预警线: 75%（黄色虚线）

  2. **费用率** (Y轴)
     - 计算公式: 费用金额 / 签单保费 × 100%
     - 业务含义: 衡量保费中被费用消耗的比例，反映经营管理效率
     - 解读指南:
       * ≤14%: 费用控制优秀
       * 14%-17%: 需关注费用管理
       * >17%: 费用率过高，需降本增效
     - 单位: 百分比（%），范围0-30
     - 应用场景: 评估后勤管理效率，识别费用浪费区域
     - 预警线: 17%（黄色虚线）

  3. **变动成本率** (气泡颜色)
     - 计算公式: 满期赔付率 + 费用率
     - 业务含义: 综合成本指标，决定公司盈利能力
     - 解读指南:
       * ≤88%（绿色）: 成本控制优秀，利润空间充足
       * 88%-91%（橙色）: 成本偏高，需要改进
       * >91%（红色）: 成本超标，亏损风险高
     - 单位: 百分比（%）
     - 应用场景: 核心管理指标，问题机构识别

  4. **保费占比** (气泡大小)
     - 计算公式: 维度签单保费 / 总签单保费 × 100%
     - 业务含义: 衡量各维度在总保费中的占比，反映业务规模
     - 解读指南: 占比越高，气泡越大，该维度越重要
     - 单位: 百分比（%）
     - 应用场景: 通过气泡大小快速识别核心业务，优先改进大气泡的成本问题

- **四象限分析框架**:
  - 左下象限（低赔付+低费用）: ✓ 优秀区域，保持现状
  - 右下象限（高赔付+低费用）: 关注赔付控制，改善承保质量
  - 左上象限（低赔付+高费用）: 关注费用控制，提升运营效率
  - 右上象限（高赔付+高费用）: ✗ 重点改进区域，双管齐下降成本

- **指标间关系**:
  - X轴（满期赔付率）+ Y轴（费用率）= 变动成本率（气泡颜色）
  - 气泡大小（保费占比）决定改进优先级：大气泡优先
  - 协同判断：成本高（红色）+ 占比大（大气泡）= 最优先改进对象
```

```markdown
### 3. 投产效率分析组合图（2025-12-22新增）

**设计目标**: 从"费用投入"和"收益产出"两个维度评估费用投入的效率和盈利质量

- **图表类型**: 双Y轴组合图
  - 左Y轴：费用产出保费比、费用产出边际贡献柱状图
  - 右Y轴：边际贡献率折线图

- **核心指标**:

  1. **费用产出保费比** (左Y轴柱状图-深蓝色)
     - 计算公式: 签单保费 / 费用额
     - 业务含义: 每1元费用投入能产生多少元的保费收入
     - 解读指南:
       * 数值越高，说明费用投入的保费产出效率越高
       * 一般行业水平：4-7元
       * <4元：费用效率低，需优化
     - 单位: 元（保留2位小数）
     - 应用场景: 评估不同维度的费用效率，识别低效率区域

  2. **费用产出边际贡献** (左Y轴柱状图-深绿色，负值红色)
     - 计算公式: 边际贡献额 / 费用额
     - 业务含义: 每1元费用投入对边际贡献（实际利润）的直接贡献度
     - 解读指南:
       * 正值越大：费用投入带来更高的实际利润贡献 ✓

     - 单位: 元（保留2位小数，负值红色加粗标注）
     - 应用场景: 评估费用投入的实际盈利效果，识别亏损业务

  3. **边际贡献率** (右Y轴折线图-深红色)
     - 计算公式: 边际贡献额 / 签单保费 × 100%
     - 业务含义: 保费收入中有多少比例转化为实际利润（扣除赔款和费用后）
     - 解读指南:
       * >20%: 盈利能力强 ✓
       * 10%-20%: 盈利能力中等
       * <10%: 盈利能力弱
       * 负值: 亏损 ✗
     - 单位: 百分比（保留1位小数）
     - 应用场景: 评估业务的整体盈利质量，判断是否需要调整策略

  4. **边际贡献额** (计算基础，不直接显示)
     - 计算公式: 签单保费 × (1 - 已报告赔款/满期保费 - 费用额/签单保费)
     - 业务含义: 保费收入扣除赔款和费用后的实际利润额
     - 应用场景: 作为计算基础，推导上述三个指标

- **指标间关系**:
  - 费用产出保费比和费用产出边际贡献都衡量"费用效率"，但侧重不同：
    * 费用产出保费比关注"收入产出"（保费规模）
    * 费用产出边际贡献关注"利润贡献"（实际盈利）
  - 边际贡献率是从保费收入的角度评估盈利能力
  - 三指标协同判断：
    * 费用产出保费比高 + 费用产出边际贡献高 + 边际贡献率高 = 高效高盈利 ✓✓
    * 费用产出保费比高 + 费用产出边际贡献低/负 = 规模大但不盈利 ✗
    * 费用产出保费比低 + 费用产出边际贡献负 = 低效且亏损 ✗✗

- **配色方案**:
  - 费用产出保费比柱：深蓝色`#0070c0`（中性，表示效率衡量）
  - 费用产出边际贡献柱：深绿色`#00b050`（正值良好），红色`#ff0000`（负值警告）
  - 边际贡献率折线：深红色`#c00000`（重点指标，利润指标用红色）

- **数据标签**:
  - 柱状图保留2位小数（元）
  - 折线图保留1位小数（%）
  - 费用产出边际贡献为负时，标签显示红色加粗
```
