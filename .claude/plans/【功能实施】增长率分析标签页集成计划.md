# 增长率分析标签页集成计划

## 目标

将增长率分析面板（GrowthAnalysisPanel）作为第四个顶级标签页集成到主仪表盘，与「综合分析」「营业货车专项」「续保专项分析」并列。

## 现状分析

### 已有组件
- ✅ `src/features/growth/components/GrowthAnalysisPanel.tsx` - 增长分析面板（完整实现）
- ✅ `src/features/growth/hooks/useGrowthAnalysis.ts` - 数据获取 Hook
- ✅ `src/shared/sql/growth.ts` - SQL 生成器（支持 YoY/MoM/YTD）

### 需要修改
- `src/features/dashboard/PremiumDashboard.tsx` - 主仪表盘容器
- `src/features/growth/components/GrowthAnalysisPanel.tsx` - Props 适配

## 实现步骤

### 步骤 1：适配 GrowthAnalysisPanel Props

**文件**: `src/features/growth/components/GrowthAnalysisPanel.tsx`

当前 Props：
```typescript
interface GrowthAnalysisPanelProps {
  filters?: FilterState;  // 旧版类型
  orgLevel3?: string;
  salesmanName?: string;
}
```

修改为：
```typescript
interface GrowthAnalysisPanelProps {
  filters: AdvancedFilterState;  // 与其他面板保持一致
}
```

**变更内容**：
1. 导入 `AdvancedFilterState` 类型
2. 从 filters 中提取 `org_level_3` 和 `salesman_name`（注意是数组类型）
3. 更新内部逻辑以支持多选过滤器

### 步骤 2：修改 PremiumDashboard.tsx

**文件**: `src/features/dashboard/PremiumDashboard.tsx`

#### 2.1 扩展标签页类型（第 43 行）

