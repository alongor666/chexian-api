**[MEDIUM] 惰性加载超时（OpenCode）：**

LazyDomainRegistry.ensureLoaded 加入 15s 超时保护：
- 超时时返回 503（"加载中，请重试"）而非无限阻塞
- 加载失败（非超时）返回 500
- 超时后 state 保持 `loading`（不标记为 failed），下次请求继续等待

实现：
```typescript
const LAZY_LOAD_TIMEOUT_MS = 15_000;

async ensureLoaded(name: string): Promise<void> {
  const entry = this.domains.get(name);
  if (!entry || entry.state === 'loaded') return;
  if (entry.state === 'loading') {
    // 超时包装：等待 loading Promise，超时则抛出 503 信号
    return Promise.race([
      entry.promise!,
      new Promise<never>((_, reject) =>
        setTimeout(() => reject(Object.assign(new Error(`Domain ${name} loading timeout`), { statusCode: 503 })), LAZY_LOAD_TIMEOUT_MS)
      )
    ]);
  }
  if (entry.state === 'failed') throw entry.error!;
  entry.state = 'loading';
  entry.promise = entry.loader()
    .then(() => { entry.state = 'loaded'; })
    .catch((err) => { entry.state = 'failed'; entry.error = err; throw err; });
  return Promise.race([
    entry.promise,
    new Promise<never>((_, reject) =>
      setTimeout(() => reject(Object.assign(new Error(`Domain ${name} loading timeout`), { statusCode: 503 })), LAZY_LOAD_TIMEOUT_MS)
    )
  ]);
}
```

LazyDomainRegistry（含 15s 超时保护，per OpenCode HIGH 关注点）:
```typescript
const LAZY_LOAD_TIMEOUT_MS = 15_000;

interface LazyDomainEntry {
  loader: () => Promise<void>;
  state: 'unloaded' | 'loading' | 'loaded' | 'failed';
  promise: Promise<void> | null;
  error: Error | null;
}

export class LazyDomainRegistry {
  private domains = new Map<string, LazyDomainEntry>();

  register(name: string, loader: () => Promise<void>): void {
    this.domains.set(name, { loader, state: 'unloaded', promise: null, error: null });
  }

  async ensureLoaded(name: string): Promise<void> {
    const entry = this.domains.get(name);
    if (!entry || entry.state === 'loaded') return;
    if (entry.state === 'loading') {
      // 等待现有 Promise（加超时保护）
      return Promise.race([
        entry.promise!,
        this.timeoutReject(name)
      ]);
    }
    if (entry.state === 'failed') throw entry.error!;
    // 首次触发加载
    entry.state = 'loading';
    entry.promise = entry.loader()
      .then(() => { entry.state = 'loaded'; })
      .catch((err) => { entry.state = 'failed'; entry.error = err; throw err; });
    return Promise.race([entry.promise, this.timeoutReject(name)]);
  }

  private timeoutReject(name: string): Promise<never> {
    return new Promise<never>((_, reject) =>
      setTimeout(() => {
        const err = new Error(`Domain ${name} loading timeout (${LAZY_LOAD_TIMEOUT_MS}ms)`);
        (err as any).statusCode = 503;
        reject(err);
      }, LAZY_LOAD_TIMEOUT_MS)
    );
  }

  isLoaded(name: string): boolean {
    return this.domains.get(name)?.state === 'loaded';
  }

  getState(name: string): string {
    return this.domains.get(name)?.state ?? 'unknown';
  }
}
```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| 路由→bootstrapper | 路由中间件（经 shared.ts）调用 ensureDomainLoaded，失败时通过 next(err) 传递，不静默吞咽 |
| 并发请求→Promise 锁 | 两个请求同时触发同一 lazy 域，只有一个 loader 实例运行，其余等待同一 Promise |
| LazyRegistry 状态机 | state 从 unloaded→loading→loaded/failed 单向流转，不得逆转（超时不改变 state） |
| ClaimsAgg→ClaimsDetail 依赖 | ClaimsAgg 最终回退路径显式先 ensureLoaded('ClaimsDetail')，防止 VIEW 未就绪时 SQL 失败 |
