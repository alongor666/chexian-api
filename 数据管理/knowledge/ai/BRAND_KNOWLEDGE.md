## 1. 解析规则

```
厂牌车型 = 品牌(中文前缀) + 型号代码(字母数字) + 车辆用途(中文后缀)

示例:
  长安SC6399D4Y客车      → 品牌=长安,   用途=客车,     大类=乘用车
  豪爵HJ150-30F两轮摩托车 → 品牌=豪爵,   用途=两轮摩托车, 大类=摩托车
  五菱LZW1029PYA货车      → 品牌=五菱,   用途=货车,     大类=货车
  梅赛德斯-奔驰GLC300越野车 → 品牌=梅赛德斯-奔驰, 用途=越野车, 大类=乘用车
```

**品牌标准化**: 大众汽车→大众, XX牌→XX, 梅赛德斯奔驰→梅赛德斯-奔驰

**用途标准化**: 纯电动轿车/两用燃料轿车→轿车, 电动两轮摩托车→两轮摩托车, 轻型载货汽车→载货汽车

**用途大类**: 乘用车(轿车/越野车/客车/MPV) | 摩托车(两轮/三轮) | 货车(载货/厢式/仓栅/牵引)

---

## 2. SQL 使用方法

```sql
-- 方法 A: JOIN 维度表（推荐，已预计算）
SELECT b.品牌, b.用途大类, SUM(p.保费)
FROM read_parquet('policy/current/*.parquet') p
JOIN read_parquet('dim/brand/latest.parquet') b ON p.厂牌车型 = b.厂牌车型
GROUP BY b.品牌, b.用途大类

-- 方法 B: 运行时提取（无维度表时的回退）
SELECT REGEXP_EXTRACT(厂牌车型, '^([\u4e00-\u9fff][\u4e00-\u9fff\-]*)', 1) AS 品牌
```
