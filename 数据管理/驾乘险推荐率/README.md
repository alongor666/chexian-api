## 🎯 项目功能

### 1. 驾乘险推介率日报
- **执行时间**: 每天 08:00 自动运行
- **数据范围**: 基于最新数据日期的连续14天
- **报告命名**: 基于数据最新日期（例如：`驾乘险推介率日报_2026-03-04.md`）
- **推送方式**: 自动推送到飞书

### 2. 机构数据拆分
- **拆分粒度**: 按三级机构（13个机构）
- **数据范围**: 基于最新数据日期的连续14天
- **文件格式**: CSV (UTF-8-BOM)
- **包含字段**:
  - 签单日期
  - 三级机构
  - 车架号
  - 保费（车险签单保费）
  - 交叉销售保费_驾意（驾意险签单保费）
