## 2. Q1 起保保单截至 Q1 末出险率（6年可比）

### 2.1 口径定义

```
分子：Q1（1-3月）起保保单中，截至同年 3/31 有出险（报案+立案）的保单数（按保单去重）
分母：Q1 起保保单总数
出险率 = 分子 / 分母
赔案件数 = 不去重，一个保单可多次出险
```

## 7. 核心结论

1. **整体向好**：2026Q1 出险率 1.35% 创6年新低，同比降 19.6%
2. **交三逆势**：非营业个人客车交三险是唯一持续上升的细分，需关注
3. **摩托车量增风险**：保单量暴增 2.4 倍，出险率同比 +40%
4. **牵引车结构性转变**：天然气占比从 4% 升至 56%，出险率是柴油的 1.7 倍
5. **天然气牵引车是 2026 年最大风险点**：满期赔付率 150%+，且量在暴增
