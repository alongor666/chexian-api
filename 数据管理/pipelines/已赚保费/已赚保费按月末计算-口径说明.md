# 已赚保费按月末计算 - 口径说明

## 文档信息

- **版本**: v3.0（滚动12个月财务口径）
- **日期**: 2026-01-17
- **状态**: 生产版本
- **用途**: 车险智慧分析系统 - 已赚保费按月末计算板块

---

## 1. 核心定义

### 1.1 已赚保费（Earned Premium）

**定义**：截至某一时点，保险公司因承担风险而实际"赚得"的保费部分。

**计算特点**：
- 保单签单时收到全部保费，但风险承担是逐日进行的
- 已赚保费随时间推移逐步增加，直到保单期满时达到全部保费
- 每张保单的已赚周期为 **365天**（一年期保单假设）

### 1.2 两种统计口径

| 口径类型 | 定义 | 应用场景 |
|---------|------|---------|
| **累计已赚保费** | 截至统计日，该保单从起保至今累计赚得的保费 | 资产负债表、准备金计算 |
| **期间已赚保费** | 在指定统计期间内新增赚得的保费 | 利润表、财务报告、KPI考核 |

**本系统采用**：**期间已赚保费（滚动12个月财务口径）**

### 1.3 保单期限假设

**统一假设**：所有车险保单视为一年期保单。
- 保险年度天数固定为 **365天**
- 止期 = 起保日 + 365天
- 不考虑闰年差异（简化处理）

## 2. 滚动12个月财务口径（核心）

### 2.1 统计窗口定义

```
┌─────────────────────────────────────────────────────────────────┐
│                    滚动12个月统计窗口                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   窗口开始日 ────────────────────────────────────► 统计日       │
│   (统计日-365天)                                    (T)         │
│                                                                 │
│   示例：统计日 = 2026-03-31                                     │
│         窗口开始 = 2025-04-01                                   │
│         统计窗口 = [2025-04-01, 2026-03-31]                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**计算公式**：
```
窗口开始日 = 统计日 - 365天 + 1天
统计窗口 = [窗口开始日, 统计日]
```

### 2.2 财务口径的核心原则

#### 原则一：首日费用按起保日归属

```
┌─────────────────────────────────────────────────────────────────┐
│ 首日费用 = P × F × α                                            │
│                                                                 │
│ 归属规则：                                                       │
│ ├── 起保日在统计窗口内 → 首日费用计入本期                        │
│ └── 起保日在统计窗口前 → 首日费用已在往期确认，本期不计入         │
└─────────────────────────────────────────────────────────────────┘
```

#### 原则二：时间分摊按期间拆分

```
┌─────────────────────────────────────────────────────────────────┐
│ 时间分摊 = P × (1-F) × (窗口内在保天数 / 365)                    │
│                                                                 │
│ 计算步骤：                                                       │
│ 1. 确定保单在保期间：[起保日, 止期]                              │
│ 2. 确定统计窗口：[窗口开始日, 统计日]                            │
│ 3. 取两者交集 = 窗口内在保期间                                   │
│ 4. 窗口内在保天数 = 交集的天数                                   │
└─────────────────────────────────────────────────────────────────┘
```

### 2.3 窗口内在保期间计算

```python
def compute_window_days(effective_date, expiry_date, window_start, window_end):
    """
    计算保单在统计窗口内的在保天数

    参数：
    - effective_date: 起保日
    - expiry_date: 止期 (= effective_date + 365天)
    - window_start: 窗口开始日
    - window_end: 统计日

    返回：窗口内在保天数（0 ~ 365）
    """
    # 窗口内在保期间的开始 = MAX(窗口开始, 起保日)
    period_start = max(window_start, effective_date)

    # 窗口内在保期间的结束 = MIN(统计日, 止期)
    period_end = min(window_end, expiry_date)

    # 如果开始晚于结束，则无在保期间
    if period_start > period_end:
        return 0

    return (period_end - period_start).days
```

## 3. 保单情形分类（六种情形）

### 3.1 情形总览图

```
时间轴：
        窗口开始日                              统计日
            │                                    │
            ▼                                    ▼
────────────┼════════════════════════════════════┼────────────►
            │◄─────── 统计窗口（365天）─────────►│
            │                                    │

六种保单情形：

情形1：窗口内起保、窗口内到期
        ├────[■■■■■■■■■]────┤
        起保              止期

情形2：窗口内起保、窗口后到期（新单未满期）
                    ├────[■■■■■■■▶
                    起保         统计日（未到止期）

情形3：窗口前起保、窗口内到期（跨窗口前边界）
    ◀■■■]────────────────┼────[■■■■■■■■]────┤
        │                起保在窗口前        止期在窗口内
        │ 首日费用已确认  │ 时间分摊：窗口开始→止期

情形4：窗口前起保、窗口后到期（横跨整个窗口）
    ◀■■■]────────────────┼════════════════════┼────[■■■▶
        │                │                    │
        │ 首日费用已确认  │ 时间分摊：整个窗口  │

情形5：窗口前已到期（不计入）
    ◀■■■■■■■■]────┤      │                    │
                止期    窗口开始              统计日
                （止期 < 窗口开始，无交集）

情形6：窗口后才起保（不计入）
                              │              ├────[■■■■▶
                              统计日          起保
                              （起保 > 统计日，尚未起保）

图例：■ = 在保期间   ◀▶ = 期间延伸   ═ = 统计窗口
```

### 3.2 六种情形详细计算

| 情形 | 起保日位置 | 止期位置 | 首日费用 | 时间分摊天数 | 说明 |
|------|-----------|---------|---------|-------------|------|
| 情形1 | 窗口内 | 窗口内 | ✅ 计入 | 起保日 → 止期 | 窗口内完整周期 |
| 情形2 | 窗口内 | 窗口后 | ✅ 计入 | 起保日 → 统计日 | 新单未满期 |
| 情形3 | 窗口前 | 窗口内 | ❌ 不计入 | 窗口开始 → 止期 | 跨前边界 |
| 情形4 | 窗口前 | 窗口后 | ❌ 不计入 | 窗口开始 → 统计日 | 横跨整个窗口 |
| 情形5 | 窗口前 | 窗口前 | ❌ 不计入 | 0（无交集） | 已完全赚完 |
| 情形6 | 窗口后 | 窗口后 | ❌ 不计入 | 0（尚未起保） | 尚未起保 |

## 4. 计算公式

### 4.1 期间已赚保费公式

```
期间已赚保费 = 首日费用部分 + 时间分摊部分

其中：
├── 首日费用部分 = P × F × α × I(起保日在窗口内)
│   └── I(条件) = 条件成立时为1，否则为0
│
└── 时间分摊部分 = P × (1-F) × (窗口内在保天数 / 365)
```

### 4.2 参数定义

| 符号 | 名称 | 计算方式 | 取值范围 |
|------|------|---------|---------|
| P | 保费金额 | 原始数据 | > 0 |
| F | 费用率 | 费用金额 / 保费 | [0, 1] |
| α | 险类系数 | 交强险=0.82，商业险=0.94 | 0.82 或 0.94 |
| T | 统计日 | 月末日期 | 日期 |
| W_start | 窗口开始日 | T - 365 + 1 | 日期 |
| E_eff | 起保日 | 原始数据 | 日期 |
| E_exp | 止期 | E_eff + 365 | 日期 |

### 4.3 窗口内在保天数计算

```python
def window_days(E_eff, E_exp, W_start, T):
    """窗口内在保天数"""
    period_start = max(W_start, E_eff)
    period_end = min(T, E_exp)
    return max(0, (period_end - period_start).days)
```

## 5. 数值案例

### 5.1 案例设置

**统计日**：2026-03-31
**窗口开始日**：2025-04-01
**统计窗口**：[2025-04-01, 2026-03-31]（365天）

**公共参数**：
- 保费 P = 10,000 元
- 费用率 F = 10%
- 险类系数 α = 0.94（商业险）

### 5.2 各情形计算

#### 情形2：新单未满期

```
保单C：起保日 2026-02-01，止期 2027-01-31
┌────────────────────────────────────────────────────────────┐
│ 时间轴：                                                   │
│ 2025-04-01 ══════════════════ 2026-02-01 ──── 2026-03-31  │
│     │                              │              │        │
│  窗口开始                        起保日         统计日     │
│                                    │◄── 59天 ──►│        │
└────────────────────────────────────────────────────────────┘

首日费用：起保日2026-02-01在窗口内 → 计入
= P × F × α = 10,000 × 0.1 × 0.94 = 940元 ✅

时间分摊天数：起保日 → 统计日 = 2026-02-01 → 2026-03-31 = 59天
= P × (1-F) × (59/365) = 10,000 × 0.9 × 0.162 = 1,455元 ✅

期间已赚保费 = 940 + 1,455 = 2,395元
```

#### 情形3：跨窗口前边界

```
保单B：起保日 2025-01-01，止期 2025-12-31
┌────────────────────────────────────────────────────────────┐
│ 时间轴：                                                   │
│ 2025-01-01 ──── 2025-04-01 ════════════ 2025-12-31 ══════ │
│     │              │                        │              │
│   起保日        窗口开始                   止期            │
│                    │◄────── 275天 ───────►│              │
└────────────────────────────────────────────────────────────┘

首日费用：起保日2025-01-01在窗口开始2025-04-01之前 → 不计入
= 0元（已在往期确认）

时间分摊天数：窗口开始 → 止期 = 2025-04-01 → 2025-12-31 = 275天
= P × (1-F) × (275/365) = 10,000 × 0.9 × 0.753 = 6,781元 ✅

期间已赚保费 = 0 + 6,781 = 6,781元
```

#### 情形4：横跨整个窗口

```
保单A：起保日 2025-07-01，止期 2026-06-30
┌────────────────────────────────────────────────────────────┐
│ 时间轴：                                                   │
│ 2025-04-01 ════ 2025-07-01 ════════════════════ 2026-03-31│
│     │              │                                │      │
│  窗口开始        起保日                           统计日   │
│     │◄───────────────── 274天 ─────────────────►│      │
│                    │                                │      │
│         窗口前（首日费用在此确认）                          │
└────────────────────────────────────────────────────────────┘

注意：起保日2025-07-01虽然在窗口内，但首日费用是在起保当日确认的！
由于我们计算的是"该窗口期间新增的已赚保费"，起保日在窗口内，首日费用应计入。

首日费用：起保日2025-07-01在窗口[2025-04-01, 2026-03-31]内 → 计入
= P × F × α = 10,000 × 0.1 × 0.94 = 940元 ✅

时间分摊天数：起保日 → 统计日 = 2025-07-01 → 2026-03-31 = 274天
= P × (1-F) × (274/365) = 10,000 × 0.9 × 0.751 = 6,758元 ✅

期间已赚保费 = 940 + 6,758 = 7,698元
```

## 8. 验证规则

### 8.1 数据验证

- [x] 费用率范围：0 ≤ F ≤ 1（异常值标记但保留）
- [x] 窗口内天数范围：0 ≤ window_days ≤ 365
- [x] 险类系数正确性：交强险=0.82，商业保险=0.94
- [x] 首日费用归属：仅当起保日在窗口内时计入

### 8.2 逻辑验证

- [x] 情形5/6：窗口前已到期或窗口后起保 → 期间已赚 = 0
- [x] 情形1/2：窗口内起保 → 首日费用计入
- [x] 情形3/4：窗口前起保 → 首日费用不计入

### 8.3 汇总验证

```
机构期间已赚保费 = Σ(各保单期间已赚保费)

验证：
- 每个机构的期间已赚保费应 ≥ 0
- 全公司期间已赚保费 = Σ(各机构期间已赚保费)
```

## 9. 与累计口径的关系

### 9.1 两种口径对比

| 维度 | 累计已赚保费 | 期间已赚保费（本系统） |
|------|-------------|---------------------|
| 定义 | 从起保至统计日的累计 | 统计窗口内新增的 |
| 首日费用 | 起保即全额计入 | 仅起保在窗口内时计入 |
| 时间分摊 | 起保日到统计日 | 仅窗口内在保天数 |
| 用途 | 准备金、资产负债 | 财务报告、KPI |

### 9.2 换算关系

```
期间已赚保费(T₁→T₂) = 累计已赚保费(T₂) - 累计已赚保费(T₁)

其中：T₁ = 窗口开始日, T₂ = 统计日
```
