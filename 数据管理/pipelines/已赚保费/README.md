## 核心口径

### 计算公式

```
earned_premium_cum = first_day_part + time_part

其中：
  first_day_part = P × F × α       (首日费用部分)
  time_part = P × (1 - F) × (E / 365)  (时间分摊部分)
```

### 参数说明

| 符号 | 含义 | 取值 |
|------|------|------|
| P | 保费 | 原始数据 |
| F | 费用率 | 费用金额 / 保费 |
| α | 险类系数 | 交强险=0.82，商业保险=0.94 |
| E | 有效天数 | min(max(T - 起保日, 0), 365) |

### 有效天数规则（监管 1/365 口径）

- 统计时点 < 起保日 → E = 0（尚未起保）
- 统计时点 - 起保日 > 365 → E = 365（一年封顶）
- 其他情况 → E = 统计时点 - 起保日
